# AgriVision: Smart Farming with Machine Learning

## Overview

AgriVision is a Machine Learning-based Crop Recommendation System designed to help farmers select the most suitable crop based on soil nutrients and environmental conditions.

## Features

* Crop Recommendation using Machine Learning
* User Authentication (Sign Up / Sign In)
* Interactive Web Interface
* Soil and Environmental Analysis
* Fast and Accurate Predictions

## Technologies Used

* Python
* Flask
* Machine Learning
* Random Forest Classifier
* HTML
* CSS

## Project Structure

├── app.py

├── requirements.txt

├── Crop_recommendationV2.csv

├── templates/

└── static/

## Installation

1. Clone the repository

```bash
git clone https://github.com/Anjum05-git/AgriVision-Smart-Farming-ML.git
```

2. Navigate to the project folder

```bash
cd AgriVision-Smart-Farming-ML
```

3. Install dependencies

```bash
pip install -r requirements.txt
```

4. Run the application

```bash
python app.py
```
